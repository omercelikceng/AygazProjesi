# Aygaz Tüp Uygulamasi

Google Firebase teknolojisi kullanarak geliştirilmiş bir android projesidir. Aygaz tüp uygulamasıdır. Kullanıcılar tüp sipariş verip alabilirler. Satıcı firma için ise de bu siparişleri görebilmeleri için bir admin paneli geliştirilmiştir. Buradan uygulamayı takip edip işlemlerini yapabilirler.
